A Next.js project to implement Figma designs into a functional web application.  
Uses **Redux Toolkit** for state management, **PrimeReact** for UI components, and **NextAuth.js** for authentication.

## Project Overview

This project implements Figma design mockups into a modern, responsive web application using Next.js 14+ App Router.  
Focus is on creating a clean authentication flow and reusable UI components.

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **State Management:** Redux Toolkit
- **UI Components:** PrimeReact with official theme
- **Authentication:** NextAuth.js
- **Styling:** PrimeReact theme with optional global CSS

## Features

- Login page
- Password reset page
- Modular, reusable UI components
- Global state management with Redux Toolkit
